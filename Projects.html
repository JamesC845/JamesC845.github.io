Linear Regression Project
In this project I use the New York Air Quality Measurements dataset to create a Linear Regression Machine Learning Algorithm. The aim of this project is to create and diagnose a model that can be used to predict changes in ozone in response to temperature and solar radiation.



Logistic Regression Project
In this project I clean a large dataset containing information on domestic US flights. I then create and diagnose a Logistic Regression Machine Learning Algorithm in order to determine the odds on flight cancellations leaving from particular US states.



Text Mining Project
In this project I use Text Mining methods to crawl and analyse internet posts to discover the current hot topics in conservation biology. This method is used to answer questions such as what habitats and species are of most concern to the public at the moment.
